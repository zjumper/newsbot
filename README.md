# newsbot
Online news chatbot api module.

## Installation

### Prerequisites

* Install NodeJS/NPM

```shell
>npm install -g grunt
```

* Install Git

### Download source code

```shell
>git clone https://github.com/zjumper/newsbot.git
```

### Build

```shell
>cd newsbot
>npm install
>npm build
```

## Run

```shell
>mkdir logs
>npm start
```
